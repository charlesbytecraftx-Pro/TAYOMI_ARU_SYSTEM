# 🚀 TAYOMI ARU Management System

Mfumo wa kisasa wa usimamizi wa wanachama, vikundi, na malipo kwa ajili ya **TAYOMI ARU**. Mfumo huu umejengwa kwa kutumia **Django Framework** na unalenga kurahisisha utendaji kazi wa kila siku.

## ✨ Sifa Kuu (Features)
* **Dashboard ya Kisasa:** Muonekano wa kuvutia (Responsive UI) kwa ajili ya Admin na Wanachama.
* **Usimamizi wa Wanachama:** Sajili na udhibiti taarifa za wanachama kwa urahisi.
* **Mfumo wa Malipo:** Fuatilia michango, malipo yanayosubiri (pending), na historia ya miamala.
* **Usimamizi wa Vikundi:** Panga wanachama katika vikundi na fuatilia ratiba zao.
* **Matukio (Events):** Kalenda ya matukio na mikutano inayokuja.

## 🛠️ Teknolojia Zilizotumika
* **Backend:** Python 3.x, Django 4.x
* **Frontend:** HTML5, CSS3, Bootstrap 5, FontAwesome icons
* **Database:** SQLite (kwa ajili ya development)
* **Animations:** Animate.css
